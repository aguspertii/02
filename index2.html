2.1- ¿qué es una url absoluta?

Las URL absolutas proporcionan la URL completa del documento, incluido el protocolo (http: // o https: //), el nombre de dominio y la ruta como necesario.

2.2- ¿qué es una url relativa?

Las URL relativas describen la ruta a un archivo relativo al actual documento. Las URL relativas se pueden usar cuando se vincula a otro documento en su propio sitio
